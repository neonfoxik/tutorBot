# TutorBot

Телеграм-бот для репетитора на Python (pyTelegramBotAPI) и Django ORM.

## Функциональность
- Бриф: последовательные вопросы с сохранением ответов в БД
- Профиль студента: ФИО, класс/курс, школа/вуз
- Оплата: заглушка вместо ЮKassa (создается запись платежа со статусом pending)
- Админ-панель бота: списки клиентов, фильтры по оплате, просмотр ожидающих платежей, рассылка напоминаний, ручное принятие/отклонение платежей
- Django admin: управление вопросами брифа, ответами и платежами

## Быстрый старт
1. Установить зависимости:
```
pip install -r requirements.txt
```
2. Настроить переменные окружения в `.env`:
```
BOT_TOKEN=xxx
OWNER_ID=123456
HOOK=https://your.host
LOCAL=True
```
3. Применить миграции:
```
python manage.py makemigrations
python manage.py migrate
```
4. Создать суперпользователя для входа в Django admin:
```
python manage.py createsuperuser
```
5. Запустить сервер и выставить webhook:
```
python manage.py runserver 0.0.0.0:8000
# перейти GET /bot/  для установки вебхука
```

## ЮKassa
Сейчас используется заглушка. При появлении данных ЮKassa:
- добавить SDK
- реализовать создание платежа и проверку статуса
- проставлять `Payment.yookassa_payment_id` и обновлять `Payment.status`
